# basic-parser

TP Final de la materia "Estructuras de Datos y Algoritmos I" de la LCC 2023.
El objetivo del trabajo es crear un parser simple que reconoce palabras de un diccionario (provisto por la cátedra) en un conjunto de frases no espaciadas, y devuelve la frase correctamente espaciada, tratando de máximizar el tamaño de cada palabra individual
